# The ANOVA Module

## Overview

<img src='inst/icons/analysis-classical-anova.svg' width='149' height='173' align='right'/>

**JASP ANOVA module** is a core module for JASP that provides tools for evaluating the difference between multiple means. The ANOVA module offers a wide range of functionalities, including frequentist and Bayesian ANOVA, repeated measures ANOVA, ANCOVA, and MANOVA.


## R Packages

<img src='https://www.r-project.org/logo/Rlogo.svg' width='100' height='78' align='right'/>

The functionality is served by several R packages

- **afex** — Convenience functions for ANOVA-like analyses ([afex on CRAN](https://cran.r-project.org/package=afex))
- **BayesFactor** — Bayesian hypothesis testing using Bayes factors ([BayesFactor on CRAN](https://cran.r-project.org/package=BayesFactor))
- **car** — Companion to Applied Regression ([car on CRAN](https://cran.r-project.org/package=car))
- **effectsize** — Indices of effect size ([effectsize on CRAN](https://cran.r-project.org/package=effectsize))
- **emmeans** — Estimated marginal means and contrasts ([emmeans on CRAN](https://cran.r-project.org/package=emmeans))
- **multcomp** — Simultaneous inference in general parametric models ([multcomp on CRAN](https://cran.r-project.org/package=multcomp))
- **restriktor** — Restricted statistical estimation and inference ([restriktor on CRAN](https://cran.r-project.org/package=restriktor))
